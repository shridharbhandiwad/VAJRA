## 🎨 Design Features

### Color Coding System

**Health Status:**
- 🟢 Excellent (90-100%): Green `#2ecc71`
- 🔵 Good (75-89%): Blue `#3498db`
- 🟡 Fair (60-74%): Yellow `#f1c40f`
- 🟠 Poor (40-59%): Orange `#e67e22`
- 🔴 Critical (<40%): Red `#e74c3c`

**Chart Aesthetics:**
- Anti-aliased rendering for smooth lines
- Animated transitions on data updates
- Gradient fills for area charts
- Semi-transparent overlays
- Professional grid lines
- Custom legends with color coding

### Theme Adaptation

**Dark Theme:**
- Background: `#1e1e1e` (Deep charcoal)
- Text: `#dcdcdc` (Light gray)
- Charts: `#2d2d2d` (Darker gray)
- Grid: `#3c3c3c` (Subtle grid)

**Light Theme:**
- Background: `#f5f5f5` (Off-white)
- Text: `#282828` (Dark gray)
- Charts: `#ffffff` (Pure white)
- Grid: `#c8c8c8` (Light gray)

## 🎯 Chart Details

### 1. Health Trend Chart (Multi-line Spline)
**Shows:** Real-time health of up to 6 components  
**Time Range:** Last 24 hours  
**Update Frequency:** Every 5 seconds  
**Features:** Smooth splines, color-coded lines, auto-scaling axes

### 2. Component Distribution (Donut Pie)
**Shows:** Breakdown by component type  
**Style:** 35% hole (donut)  
**Features:** Interactive slices, percentage labels, color variety

### 3. Subsystem Performance (Bar Chart)
**Shows:** Average health per subsystem  
**Orientation:** Vertical bars  
**Features:** Uniform color, value labels, category axis

### 4. Health Distribution (Stacked Area)
**Shows:** Components in Critical/Warning/Good zones  
**Style:** Semi-transparent fills  
**Features:** Color zones, count visualization, time-based

### 5. Message Frequency (Scatter Plot)
**Shows:** Message events over 6 hours  
**Style:** 8px markers  
**Features:** Component indexing, event clustering, temporal analysis

### 6. Telemetry Streams (Multi-line)
**Shows:** Temperature, Power, Signal Strength  
**Metrics:** 3 simultaneous streams  
**Features:** Color-coded lines, 100-minute window, real-time

### 7. Alert History (Grouped Bar)
**Shows:** Critical vs Warning alerts per hour  
**Time Range:** Last 12 hours  
**Features:** Dual-series, color-coded severity, hourly breakdown

### 8. Component Comparison (Horizontal Bar)
**Shows:** Current health of up to 8 components  
**Orientation:** Horizontal  
**Features:** Easy ranking, consistent color, component names

### 9. System Efficiency (Gradient Area)
**Shows:** Overall system performance  
**Style:** Purple gradient fill  
**Features:** Smooth curve, 100-minute history, aesthetic appeal

### 10. Uptime Analysis (Stacked Bar)
**Shows:** Uptime vs Downtime per component  
**Style:** 100% stacked bars  
**Features:** Green/Red coding, percentage display, 6 components

## 🚀 Quick Start

### Build Requirements
```bash

sudo apt-get install qt5-default qtcharts5-dev

# Build project
cd UnifiedApp
qmake UnifiedApp.pro
make
```

### Launch
```bash
./UnifiedApp
```

2. Click **"📊 VIEW DASHBOARD"** in toolbar
3. Explore charts and KPIs!
